Što radimo s time? Pejstam ono što je Ben predložio: 

Corpus-based frequency/time-series analysis (Croatian): we will extract and track occurrences from SketchEngine, primarily the hrWac web corpus (and relevant subcorpora where appropriate).


News/event-driven media sampling: we will collect and structure additional recent media coverage using EventRegistry (querying for the relevant slurs and associated topics) to anchor linguistic spikes to concrete, time-stamped events.


Event/spike alignment: we will compare slur-frequency spikes with polarizing events using event timelines (EventRegistry) and complementary public attention indicators (e.g., Google Trends-style spike checks for the key terms).

Lexical analysis via similarity methods: we will run a lexical-semantic similarity layer (embedding/distributional similarity) to capture variants, near-synonyms, and contextual clustering of the slurs, following standard lexical similarity methodology used in our prior lexical-semantic work (Perak et al., in preparation).
